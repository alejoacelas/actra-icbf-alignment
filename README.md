# ACTRA × ICBF strategic alignment

[ACTRA.ngo](https://actra.ngo) is preparing for a call with ICBF (Instituto Colombiano de Bienestar Familiar). We crawled ~100 pages from icbf.gov.co using Firecrawl and analyzed them with Claude Code to find where ACTRA's group-CBT crime prevention program fits within ICBF's structure.

- **[informe-alineacion-actra-icbf.md](informe-alineacion-actra-icbf.md)** — Strategic alignment report in Spanish, with direct quotes, terminology mappings, and ready-to-use phrases for the call.
- **[replication-guide.md](replication-guide.md)** — How this was done, in English.
